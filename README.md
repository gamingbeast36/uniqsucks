# uniqsucks

Have you ever had to use uniq to show duplicate lines ughh it's a pain. 

Therefore, we have uniqsucks a very simple utility
written in golang to show duplicate lines in a file and to make a file without the duplicate lines.

To download just grab a binary from releases or git clone the repository to compile and use it yourself

The command below is all you need to know to use uniqsucks
> uniqsucks --help
